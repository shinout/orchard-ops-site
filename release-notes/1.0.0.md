Orchard Ops 1.0.0 — the first stable release.

Schedule scripts and commands on your Mac with a visual timeline. Built on launchd, no background daemon required.

**Highlights:**

- Visual day timeline showing all scheduled jobs at a glance
- Flexible scheduling: daily, weekly, monthly, interval, or run-at-load
- Wake-before-run: your Mac wakes up to run the job, then sleeps again
- Built-in CLI: manage tasks from the terminal with `orchard-ops`
- Import/export tasks as JSON for backup and sharing
- Automatic updates via Sparkle
- Also available via Homebrew: `brew install orchardworks/tap/orchard-ops`

Free tier includes 3 tasks. Purchase a license at [orchard-ops.com](https://orchard-ops.com) to unlock unlimited.
