Frequently Asked Questions

How do I install Orchard Ops?

Download the DMG, open it, and drag Orchard Ops to your Applications folder. On first launch, macOS may ask you to confirm — right-click the app and choose "Open" if needed. You can also install via Homebrew: brew install orchardworks/tap/orchard-ops

Why does it ask for administrator permission?

Orchard Ops uses pmset to wake your Mac before scheduled tasks. This requires administrator privileges. It's only used when you enable the "Wake Mac for this task" option.

My task didn't run. What should I check?

Make sure the task is enabled (toggle in the sidebar). Check the task detail view for run history and error logs. If your Mac was asleep, ensure "Wake Mac for this task" is turned on.

Does it work on MacBook?

Yes. Orchard Ops works on any Mac. The wake-from-sleep feature works best when plugged into power, but everything else works anywhere.

What's the difference between Free and Pro?

Only the number of tasks. Free supports up to 3 scheduled tasks. Pro unlocks unlimited tasks for a one-time $15 purchase. All features are available in both.

How do I set up Gmail reminders?

When creating a new task, click "Gmail Template". You'll need a Google App Password — the setup wizard walks you through it. Your password is stored securely in macOS Keychain.

How does it update?

Automatic updates via Sparkle. You'll get a notification when a new version is available — one click to update.

Will it affect my existing cron jobs or launchd tasks?

No. Orchard Ops creates its own launchd jobs and never modifies existing ones. You can browse system jobs in the app, but they're read-only.

How do I uninstall?

Drag Orchard Ops from Applications to the Trash. The app's launchd jobs will stop running automatically. To clean up config files, remove ~/Library/Application Support/OrchardOps.
