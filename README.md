# Orchard Ops

A native macOS task scheduler for developers and power users. Schedule CLI commands, scripts, and Apple Shortcuts without writing launchd plists by hand.

**Website:** [orchard-ops.com](https://orchard-ops.com)

## Features

- **Flexible Scheduling** — Daily, hourly, monthly, or at login
- **Gmail Reminders** — Send recurring emails with a built-in Gmail template (App Password stored in Keychain)
- **Day Timeline** — Visualize all tasks on a 24-hour timeline
- **Shortcuts Integration** — Run Apple Shortcuts on a schedule
- **Wake Mac** — Wake from sleep before a task runs
- **System Job Browser** — Inspect existing launchd jobs on your system
